# Platxa Frontend Agent - Video Tutorials

This directory contains scripts and documentation for video walkthroughs of key Platxa Frontend Agent features.

## Tutorial Series

### 1. [Setup Walkthrough](./01-setup-walkthrough.md)
**Duration:** ~10 minutes
**Level:** Beginner

Learn how to install and configure Platxa Frontend Agent in your project. Covers:
- Installation via npm/pnpm
- Basic configuration
- First component generation
- Verifying the setup

### 2. [Custom Brand Creation](./02-custom-brand-creation.md)
**Duration:** ~15 minutes
**Level:** Intermediate

Create your own brand kit with custom colors, typography, and design tokens. Covers:
- Understanding the brand kit structure
- Defining semantic colors
- Setting up light/dark modes
- Generating CSS variables

### 3. [Advanced Features](./03-advanced-features.md)
**Duration:** ~20 minutes
**Level:** Advanced

Master advanced features for production applications. Covers:
- Framework integrations (Next.js, Vite, Remix, Astro)
- Build tool plugins (PostCSS, ESLint, Webpack)
- Runtime theme switching
- Performance optimization

## How to Use These Tutorials

Each tutorial is structured as a step-by-step guide with:
- **Timestamps** for video chapter markers
- **Code examples** you can copy/paste
- **Expected outputs** to verify your progress
- **Troubleshooting tips** for common issues

## Prerequisites

Before starting, ensure you have:
- Node.js 18+ installed
- A package manager (npm, pnpm, or yarn)
- A code editor (VS Code recommended)
- Basic knowledge of React and TypeScript
